# QA Checklists

## Keyframe QC (before Kling generation)

- [ ] Same face identity as Character Passport
- [ ] Same eye structure, nose, mouth
- [ ] Same ethnicity and age band
- [ ] Correct emotion for this scene
- [ ] Correct camera framing
- [ ] Correct location/environment
- [ ] No random extra people
- [ ] No identity drift from previous scene keyframe
- [ ] Wardrobe consistent with storyboard

## Clip QC (after Kling generation)

- [ ] Same person as keyframe (no identity drift)
- [ ] Lip movement matches spoken text
- [ ] Emotion feels natural and correct
- [ ] No visual glitches or artifacts
- [ ] No random people appearing
- [ ] Motion looks realistic (no uncanny movements)
- [ ] Audio quality acceptable
- [ ] Duration matches scene spec
- [ ] Camera framing correct

## Final Video QC (after Remotion render)

- [ ] All scenes in correct order
- [ ] Transitions smooth and clean
- [ ] Subtitles readable on mobile
- [ ] Subtitle timing matches speech
- [ ] Music volume appropriate (not overpowering voice)
- [ ] Music ducks properly under speech
- [ ] Website insert looks natural (phone frame aligned)
- [ ] TICKETS mention feels organic, not forced
- [ ] Total duration 30-45 seconds
- [ ] No black frames or gaps
- [ ] Aspect ratio correct (9:16)
- [ ] Would you actually post this? (final gut check)
